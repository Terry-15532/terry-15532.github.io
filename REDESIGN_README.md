# 个人网站重构 - 鹰角风格设计

## 🎨 设计概念

本次重构采用**扁平化现代风格**并融入**二次元元素**，灵感来源于鹰角网络（Hypergryph）的UI设计，特别是《明日方舟》（Arknights）的视觉风格。

## ✨ 主要特点

### 1. 视觉设计
- **深色主题**：采用深色背景系统，营造科技感氛围
- **青蓝强调色**：使用 #00d9ff 作为主要强调色，带来科技感和未来感
- **几何装饰元素**：斜角切割、装饰性线条、角落装饰框
- **动态网格背景**：带有动画效果的网格背景
- **渐变光晕**：柔和的径向渐变创造景深

### 2. 交互动画
- **流畅的页面切换**：优化的淡入淡出动画（400ms）
- **悬停效果**：卡片抬升、边框发光、按钮动画
- **渐进加载**：内容按顺序淡入，增强视觉层次
- **响应式反馈**：所有可点击元素都有明确的视觉反馈

### 3. 组件系统

#### 导航栏（Header）
- 半透明毛玻璃效果
- 顶部青色渐变装饰线
- 下划线激活指示器
- 悬停时文字高亮和抬升效果

#### 按钮系统
- 主按钮：带边框和悬停光晕效果
- 标签按钮：扁平化设计，带边框
- 统一的过渡动画

#### 卡片系统
- 半透明背景 + 毛玻璃效果
- 顶部渐变线装饰
- 右上角斜角装饰
- 悬停时抬升和边框发光

#### 项目展示
- 大图展示 + 详细信息布局
- YouTube视频嵌入
- 标签系统（技术栈、团队规模等）
- 金奖项目特殊高亮

### 5. 响应式设计
- 移动端优化布局
- 导航栏自适应

- 触摸友好的交互区域

## 🔧 自定义指南

### 修改主题颜色

在 `CommonStyles.css` 的 `:root` 部分修改：

```css
:root {
    --color-accent-primary: #00d9ff;  /* 改为你喜欢的颜色 */
    --color-accent-secondary: #00a8cc;
}
```

### 修改动画速度

```css
:root {
    --transition-fast: 150ms;    /* 快速动画 */
    --transition-normal: 300ms;  /* 正常速度 */
    --transition-slow: 500ms;    /* 慢速动画 */
}
```

### 修改间距系统

```css
:root {
    --spacing-xs: 4px;
    --spacing-sm: 8px;
    --spacing-md: 16px;
    --spacing-lg: 24px;
    --spacing-xl: 32px;
    --spacing-2xl: 48px;
    --spacing-3xl: 64px;
}
```

## 💡 使用提示

1. **保留备份文件**：首次部署后测试一段时间，确认无问题后再删除 `*_OLD` 文件
2. **渐进增强**：如果遇到兼容性问题，可以添加浏览器前缀
3. **性能优化**：网格背景动画在低端设备上可能影响性能，可以通过 CSS 禁用

## 🎨 设计哲学

- **Less is More**：通过留白和简洁的元素创造优雅感
- **一致性**：所有组件遵循统一的设计语言
- **可访问性**：高对比度文字，清晰的交互反馈
- **性能第一**：优化动画性能，避免不必要的重绘
