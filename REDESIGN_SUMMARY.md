## 🔧 自定义建议

### 修改主题色
在 `CommonStyles.css` 第7-8行修改：
```css
--color-accent-primary: #00d9ff;  /* 主强调色 */
--color-accent-secondary: #00a8cc; /* 次强调色 */
```

### 调整动画速度
在 `CommonStyles.css` 第31-33行修改：
```css
--transition-fast: 150ms;
--transition-normal: 300ms;
--transition-slow: 500ms;
```

### 修改间距
在 `CommonStyles.css` 第23-29行调整间距系统
