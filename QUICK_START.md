# 🚀 快速启动指南

## 查看重构后的网站

### 方法1：直接打开（推荐）
1. 找到 `index.html` 文件
2. 双击打开（会在默认浏览器中打开）
3. 浏览所有页面查看新设计

### 方法2：使用Live Server（推荐开发时使用）
1. 在VS Code中安装 "Live Server" 扩展
2. 右键点击 `index.html`
3. 选择 "Open with Live Server"
4. 自动在浏览器中打开并支持热重载

### 方法3：使用Python服务器
```bash
# Python 3
cd "e:\Github Repositories\terry-15532.github.io"
python -m http.server 8000

# 然后在浏览器打开: http://localhost:8000
```

### 方法4：使用Node.js服务器
```bash
# 安装 http-server (只需一次)
npm install -g http-server

# 启动服务器
cd "e:\Github Repositories\terry-15532.github.io"
http-server

# 在浏览器打开显示的地址
```

## 🛠️ 常见问题

### Q: 某些效果看不到？
A: 确保使用现代浏览器（Chrome/Edge/Firefox最新版）

### Q: 毛玻璃效果不显示？
A: backdrop-filter在某些浏览器需要启用实验性功能，或使用Chrome/Safari

### Q: 想要修改颜色？
A: 编辑 `CommonStyles.css` 的 `:root` 部分

### Q: 动画太快/太慢？
A: 修改 `CommonStyles.css` 中的 `--transition-*` 变量

### Q: 想恢复旧版本？
A: 所有原文件已备份为 `*_OLD.html`，重命名即可恢复
